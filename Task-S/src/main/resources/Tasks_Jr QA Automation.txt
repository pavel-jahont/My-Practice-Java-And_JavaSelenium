========== Вводная информация ==========
Разместите проект например в бесплатном репозитории bitbucket.org, так будет проще нам обмениваться результатами.
Для написания и запуска тестов используйте Intellij IDEA, JAVA 8, Selenium WebDriver, TestNG, используйте сборщик Maven или Gradle.
Для вывода логов в консоль и формирования лог файла используйте Log4J.
Вы должны овладеть PageObject pattern и показать ваш опыт в ООП.
Будет очень хорошо если ваши тесты будут Кроссбраузерными (FireFox, IE, Chrome), но по началу достаточно FireFox.
Все задания выполнить в одном проекте с возможностью запускать их кажое по отдельности и все вместе.

Task 4

Напишите программу на Java, которая подключается к FTP серверу (например ftp.byfly.by) можно использовать анонимное
подключение (anonymous/anonymous). Программа должна получить список директорий и перейти в каждую из директорий
первого уровня. Затем попробовать создать директорию, в случае успеха удалить директорию и отключиться от сервера.
 
Примечание: В задании нужно использовать java.net.Socket. Данное задание также можно выполнить в виде юнит тестов.
